# 🍔 iHungry Burger Shop - OOP Swing Application

A Java Swing-based burger shop management system built using Object-Oriented Programming principles.

## 📋 Project Overview

This is a desktop application developed in Java using Swing GUI framework for managing a burger shop's operations. The project demonstrates the implementation of OOP concepts in a real-world scenario.

## 🛠️ Technologies Used

- **Language:** Java
- **GUI Framework:** Java Swing
- **IDE:** NetBeans
- **Build Tool:** Apache Ant (build.xml)

## 📁 Project Structure

```
OOP-Swing-iHungry-BurgerShop/
│
├── src/                    # Source code directory
├── lib/                    # External libraries
├── nbproject/             # NetBeans project configuration
├── build.xml              # Ant build configuration
└── manifest.mf            # JAR manifest file
```

## 🚀 Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or higher
- NetBeans IDE (recommended)
- Apache Ant (for building)

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/PasinduOG/OOP-Swing-iHungry-BurgerShop.git
   ```

2. **Open in NetBeans:**
   - Launch NetBeans IDE
   - File → Open Project
   - Navigate to the cloned repository folder
   - Select and open the project

3. **Build the project:**
   ```bash
   ant compile
   ```

4. **Run the application:**
   ```bash
   ant run
   ```

## 🎯 Features

- **Object-Oriented Design:** Implements core OOP principles including encapsulation, inheritance, and polymorphism
- **Swing GUI:** User-friendly desktop interface
- **Burger Shop Management:** Complete system for managing burger shop operations

## 🏗️ Architecture

This project follows Object-Oriented Programming principles:

- **Encapsulation:** Data and methods are properly encapsulated within classes
- **Inheritance:** Utilizes class inheritance for code reusability
- **Polymorphism:** Implements method overriding and overloading
- **Abstraction:** Abstract classes and interfaces for better design
